# OTT Platforms Video Forward/Rewind with Arrow Keys

Take back the control to forward and rewind live sport video playback on OTT platforms like Hotstar, SonyLiv, etc with keyboard arrow keys.

## Installation

Download the extension from [here](https://github.com/pea1bee/ott-live-sport-rewind/archive/refs/heads/main.zip)

- Unzip the file
- Go to  ```chrome://extensions/```
- Click on "Load unpacked"
- Select the unzipped folder
