# OTT Live Sport Video Forward/Rewind with Arrow Keys

A Chrome extension that lets you seek forward and backward during live sport streams on OTT platforms using keyboard arrow keys.

> **Update:** Now supports going back and forward with Arrow Keys on IPL streams on Hotstar!

## Features

- Use **← / →** arrow keys to rewind and fast-forward live sport video
- Works on Hotstar and SonyLiv

## Installation

1. [Download the extension](https://github.com/pea1bee/ott-live-sport-rewind/archive/refs/heads/main.zip)
2. Unzip the downloaded file
3. Open `chrome://extensions/` in your browser
4. Enable **Developer mode** (toggle in the top-right corner)
5. Click **Load unpacked** and select the unzipped folder
